# 🔐 Face Recognition Smart Door System with AI Assistant

A smart security system that uses facial recognition for door access, auto-approval mechanisms, and an AI-powered assistant to monitor and interact with logs. Ideal for enhancing security in homes, offices, or institutions.

---

## 📸 Features

- ✅ Real-time facial recognition using a webcam
- ✅ Automatic door unlocking for known users
- 🔒 Admin approval for unknown users (via email)
- 📧 Email notifications for unknown person detection
- 📊 Flask-based web interface for log management
- 🧑‍💼 Multi-user roles (Admin & Security Officer)
- 🔐 Two-Factor Authentication (2FA) for secure login

---


## 🔐 Security Features

- Two-Factor Authentication (email-based)
- Role-based dashboards (Admin vs Security Officer)
- Logs all login attempts & security events
- Email alerts for unknown access
- Admin approval for new face detections

---

## 🧪 Tech Stack

- Python, OpenCV, face_recognition
- Flask + Streamlit (UI & backend)
- Pandas, Numpy, Scikit-learn
- LLM (LLaMA/Mistral via Ollama or Hugging Face)
- SMTP for Email alerts
- MongoDB/MySQL for user data and logs

---

## 📌 Future Improvements

- Add mobile notification support
- Train face recognition on custom datasets
- Add automatic night mode camera
- Integrate with IoT devices for door locking
- Replace CSV with database for encoding storage
